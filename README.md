## Notes

### 1. Django setup
```bash
pip install django
django-admin startproject myproject
cd myproject
python manage.py runserver
```

### 2. Python & Django files
- `manage.py` — CLI tool to run commands
- `settings.py` — all project config lives here
- `urls.py` — URL routing
- `views.py` — logic that handles requests
